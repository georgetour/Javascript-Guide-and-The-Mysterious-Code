Just put the external script code at the head of the body with defer attribute.

DOM - Document Object Model

Probably you have already heard with javascript you will be able to manipulate the DOM and do wonderful stuff.

But what's the point of manipulating something if you don't know what it is?

When a web page is loaded it creates a Document Object Model of the page.

It's a world standard that was created from W3C so every browser can show you html and xml content.

The objects are what we have in our document from head, body, elements and they have properties, methods, events.

Imagine the DOM as the tree which the document has roots and branches as you can see here.

The model defines those standards. With javascript you can change anything in the DOM.

That's why it is called the DOM. You have the document, objects in it and the model. So they named it DOM - Document Object Model.

Everything in the DOM is a node.

To see DOM properties, methods etc in chrome you can see them in properties tab at right when you have chosen an element, in mozilla with firebug or the add-on DOM inspector a tab that's called DOM.
